# Loon 插件目录

本目录收录可直接导入 Loon 的插件配置与对应外部脚本。

## 插件列表

| 插件 | 路径 | 主要功能 |
| --- | --- | --- |
| 123云盘去广告 | `123Pan/123pan.plugin` | 123云盘 App 与网页分享页广告净化，并保留既有外部解除下载限制规则。 |
| 汽车之家去广告 | `CarHome/CarHome.plugin` | 清理汽车之家开屏弹窗、运营卡、信息流广告、社区横幅和二手车插卡。 |
| 音乐去广告 | `MusicAd/音乐去广告.plugin` | 拦截 QQ 音乐互动广告和波点音乐开屏广告。 |

## 使用说明

每个插件目录内包含独立 README、CHANGELOG、`.plugin` 配置和必要的外部脚本。导入前请确认 Loon MITM 证书已安装并信任；更新插件后建议完全关闭对应 App 后重新打开测试。

## 维护规则

- 修改插件行为时，同步更新对应目录的 `README.md` 与 `CHANGELOG.md`。
- 外部脚本应保持最小改写范围，只处理已确认的广告或推广字段。
- 不要提交临时抓包文件、测试输出或无用途占位文件。
