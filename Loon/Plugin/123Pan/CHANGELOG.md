# Changelog

## v1.0.1 - 2026-06-23

- 将插件重写段名从 `[Rewrite]` 调整为 Loon 常用的 `[URL Rewrite]`，保持原有广告图片与网页广告脚本拦截规则不变。
- 修正插件描述，明确广告净化规则之外仍保留既有外部“解除下载限制”脚本。
- 增加外部脚本对异常运行环境的 `$request/$response` 容错，不改变广告字段处理逻辑。
- 修正文档乱码。

## v1.0.0

- 添加 123云盘去广告插件：净化广告配置、清空广告资源、关闭活动弹窗、清空公告推广、清理用户中心推广文案。
- 网页分享页注入有限 CSS 隐藏 advert/banner/promotion/popup 等广告容器，并移除 app_ad_ 开头的图片 URL。
- 插件入口使用 GitHub Raw 外部脚本支持（script-path 指向仓库中的 123pan-ad.js）。
- 广告净化脚本不包含或修改下载限制、流量限制、会员权益或下载鉴权相关逻辑。
