# 金融政务防代理检测增强版 (大满贯版)

## 📝 简介
全面跳过国内所有主流银行、云闪付及政务 App 的代理检测与 MITM 解密。有效解决由于开启代理导致的网络连接中断、无法登录、页面加载失败、支付报错等问题。

### ✨ 支持/覆盖范围
* **支付与清算机构**：银联 / 云闪付、网联清算等。
* **六大国有商业银行**：农业银行、工商银行、建设银行、中国银行、交通银行、邮政储蓄银行。
* **十二家股份制商业银行**：招商银行、浦发银行、中信银行、光大银行、平安银行等。
* **地方性城商行**：北京银行、上海银行、宁波银行等。
* **政务与公共服务**：交管 12123、个人所得税、国家医保服务平台、各地政务 App 等（基于 `.gov.cn` 顶级域名全面覆盖，已根据需求排除 12306）。

## ⚠️ 注意事项与排障

1. **避免插件冲突**：如果您之前安装过其他开发者编写的“跳过代理检测”、“去代理”插件，**请先将其停用或删除**，以免产生规则冲突。
2. **禁止开启去广告脚本**：对于农业银行、云闪付等金融类 App，如果您启用了“去开屏广告”、“UI 净化”等带有 `http-response` 或 `http-request` 的重写（Rewrite）脚本，系统仍会拦截解密从而导致断网报错。**请务必关闭这些针对金融政务类 App 的去广告插件。**
3. **彻底重启 App**：导入启用本插件后，**请务必在多任务界面将需要使用的银行/政务 App 彻底划掉（杀掉后台）**，再重新打开，即可享受稳定直连的原生网络环境。
