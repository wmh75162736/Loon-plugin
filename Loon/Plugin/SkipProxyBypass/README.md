# 金融政务防代理检测增强版 (大满贯版)

## 📝 简介
全面跳过国内所有主流银行、云闪付及政务 App 的代理检测与 MITM 解密。有效解决由于开启代理导致的网络连接中断、无法登录、页面加载失败、支付报错等问题。

### ✨ 支持/覆盖范围
* **支付与清算机构**：银联 / 云闪付、网联清算。
* **六大国有商业银行**：农业银行、工商银行、建设银行、中国银行、交通银行、邮政储蓄银行。
* **十二家股份制商业银行**：招商银行、浦发银行、中信银行、光大银行、华夏银行、民生银行、广发银行、兴业银行、平安银行、浙商银行、渤海银行、恒丰银行。
* **地方性城商行**：北京银行、上海银行、江苏银行、宁波银行。
* **政务与公共服务**：交管 12123、个人所得税、国家医保服务平台、各地政务 App 等（基于 `.gov.cn` 顶级域名全面覆盖，已根据需求排除 12306）。
