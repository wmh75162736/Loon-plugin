# CarHome 去广告

## 更新信息

- 当前版本：v1.2.2
- 更新日期：2026-06-20
- 适用客户端：Loon（iOS）

## 文件

| 文件 | 用途 |
| --- | --- |
| `CarHome.plugin` | Loon 插件配置，包含脚本、广告接口空响应、素材拦截与 MITM。 |
| `CarHome.js` | 只清理已确认的运营卡 JSON，不修改正常页面数据。 |
| `CHANGELOG.md` | 版本更新记录。 |

## 功能范围

- 清理抓包确认的首页运营弹窗和悬浮运营卡。
- 清理广告创意接口、运营导航和广告服务接口。
- 清理“我的”页面独立请求的推广卡，例如购车补贴和顶部推广图。
- 拦截本次抓包中的 `adm2.autoimg.cn` 广告素材。
- 清理社区顶部运营横幅；保留话题、分类和正常社区内容。
- 清理选车页的品牌补贴标签、免费查成交价卡和“限时补贴”直播悬浮卡。
- 清理品牌车型页顶部的推荐车型推广位，以及“专业主播帮你选好车”直播卡。
- 保留原版的独立直播响应处理：直播浮窗和报价页直播内容。
- 清理首页信息流 `adlist` 广告，以及二手车页面的推广轮播和视频插卡。

## 保留内容

- “我的”页面核心接口：`/platform/carserver/carcard/mycardv8`。
- 车辆、账号、服务入口、正常资讯和社区内容。
- 二手车真实 `carlist` 车辆列表。
- 启动配置、域名配置和网络配置接口。

## 使用方法

1. 删除或关闭旧的“汽车之家去广告”Surge 模块，避免旧规则继续改写 `mycardv8`。
2. 上传本目录全部文件至 GitHub：`Loon/Plugin/CarHome/`。
3. 在 Loon 导入 `CarHome.plugin`，并确认 Loon MITM 证书已安装和信任。
4. 退出汽车之家 App 后重新打开测试。

## 更新日志规则

后续每次修改均在 `CHANGELOG.md` 顶部新增版本号、更新日期和具体变化；若插件配置没有变化，只更新外部脚本与更新日志。
